# 🚨 Emergency Response System

## Project Overview
The **Emergency Response System** is a web-based application designed to help manage and respond to emergency incidents efficiently. It allows users to report incidents, track updates, assign responders, and manage roles with secure authentication and authorization.

This system improves coordination between responders and administrators by providing real-time updates and structured incident management.

## Features

### Authentication & Authorization
- User registration and login system
- Role-based access control (Admin, Dispatcher, Citizen)
- Secure authentication using ASP.NET Core Identity
- Only authorized users can access or modify specific data

---

### User Management
- Manage user roles and permissions
- Admin can assign roles (Admin / Dispatcher / Citizen)
- Control access to features like editing, deleting, and viewing data

---

### Incident Management
- Create and view emergency incidents
- Store details like title, description, and location
- Track all incidents in the system

---

### Responders System
- Assign responders to incidents
- Manage responder details (name, occupation)
- Link responders to specific incidents

---

### Incident Updates & Status Tracking
- Add updates to incidents
- Track status changes over time
- Maintain history of incident progress

---

### Database Integration
- Uses Entity Framework Core for data management
- SQL Server / LocalDB support
- Migrations for database version control
